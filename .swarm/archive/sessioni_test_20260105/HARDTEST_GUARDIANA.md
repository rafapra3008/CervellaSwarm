# HARDTEST: Guardiana Qualita

**Assegnato a:** cervella-guardiana-qualita
**Priorita:** Alta
**Stato:** ready

## Obiettivo

Sei la Guardiana della Qualita. Devi fare review del report creato dal backend worker.

## Task

1. Leggi il file `.swarm/tasks/TASK_TEST_BACKEND_output.md`
2. Valuta la qualita del report:
   - E completo?
   - I suggerimenti sono validi?
   - Manca qualcosa?
3. Scrivi il tuo verdetto

## Output Atteso

Scrivi in `HARDTEST_GUARDIANA_output.md`:
- APPROVED o NEEDS_WORK
- Motivazione (2-3 righe)
- Eventuali note

## Criteri di Successo

- [ ] Report letto
- [ ] Verdetto dato
- [ ] Motivazione chiara
