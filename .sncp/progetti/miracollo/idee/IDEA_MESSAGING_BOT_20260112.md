## Use Cases Identificati

### 1. Revenue Management (già pensato)
```
Bot → Manager: "Domani 5 camere vuote. Attivo promo -15%? Rispondi SI"
Manager → Bot: "SI"
Bot: "Fatto! Promo attiva fino a domani ore 18"
```

### 2. Previsioni Pasti (NUOVO!)
```
Chef → Bot: "Quanti al pranzo oggi?"
Bot → Chef: "12 ospiti confermati + 3 probabili (check-in ore 11)"
Bot → Chef: "2 con allergie: Camera 101 (glutine), Camera 205 (lattosio)"
```

### 3. Eventi del Giorno
```
Bot → Reception (ore 7): "Oggi:
- Check-in: 5 (primo ore 14)
- Check-out: 3 (ultimo ore 11)
- Colazioni previste: 18
- Evento: Cena aziendale ore 20 (25 pax)"
```

### 4. Dettagli Importanti Real-time
```
Bot → Housekeeping: "Camera 301 check-out completato, ready for cleaning"
Bot → Manager: "Recensione 5 stelle appena ricevuta!"
Bot → Reception: "Ospite VIP in arrivo tra 30 min (repeat guest)"
```

### 5. Query Interattive (BIDIREZIONALE!)
```
Receptionist → Bot: "Status camera 205?"
Bot: "Camera 205: Occupata, check-out domani, 2 adulti, no richieste speciali"

Manager → Bot: "Revenue oggi?"
Bot: "Revenue oggi: 2.340 EUR (+12% vs stesso giorno anno scorso)"
```
