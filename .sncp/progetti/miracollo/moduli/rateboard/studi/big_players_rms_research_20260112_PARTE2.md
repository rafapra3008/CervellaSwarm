## 4. Must Have Features (2026)

### 4.1 Table Stakes (Non Negoziabili)

Queste feature sono presenti in **TUTTI** i RMS moderni. Se non le hai, non competi:

1. **Dynamic Pricing Algorithms**
   - Adjust automatico basato su demand
   - Real-time o near real-time
   - Rule-based minimo, ML-based preferito

2. **Demand Forecasting**
   - Historical data analysis
   - Seasonal patterns
   - Booking windows & curves
   - Pick-up monitoring

3. **Competitor Rate Shopping**
   - Daily scraping minimo
   - Multiple competitors tracking
   - Rate parity monitoring
   - OTA vs direct comparison

4. **PMS Integration**
   - Bi-directional sync
   - Real-time (o near real-time) updates
   - Inventory management
   - Booking data flow

5. **Channel Manager Integration**
   - Multi-channel distribution
   - Rate/availability sync
   - Stop-sell automation
   - Overbooking prevention

6. **Reporting & Analytics**
   - RevPAR, ADR, Occupancy tracking
   - Performance dashboards
   - Historical comparison
   - Export capabilities

7. **Cloud-based Architecture**
   - SaaS delivery
   - No on-premise richiesto
   - Multi-device access
   - Automatic updates

8. **Open Pricing Capability**
   - Price by room type independently
   - Price by channel independently
   - No forced BAR lockstep
   - Flexible pricing rules

    - Auto-pilot mode disponibile
    - Manual override possibile

11. **Data Visualization**
    - Graphical reports (non solo tabelle)
