## TECHNICAL DEEP DIVE

### Architecture Patterns

**CENTRALIZED AGGREGATOR** (Delta, CoinStats):
```
User ‚Üí API Keys ‚Üí Tracker Backend ‚Üí Exchange APIs ‚Üí Real-time sync
```
- Pro: Auto-sync, accurate
- Con: Security risk (API keys), trust needed

**MANUAL ENTRY** (CoinGecko, CoinMarketCap):
```
User ‚Üí Manual input ‚Üí Local storage ‚Üí Price API ‚Üí Display
```
- Pro: No API keys, sicuro
- Con: Tedioso, prone to errors

**WALLET-FIRST** (Zerion):
```
User Wallet ‚Üí On-chain read ‚Üí Multi-chain indexer ‚Üí Dashboard
```
- Pro: Permissionless, privacy
- Con: CEX assets invisible

**ON-CHAIN INTELLIGENCE** (Nansen):
```
Blockchain nodes ‚Üí Indexer ‚Üí Label database ‚Üí Analytics engine ‚Üí Insights
```
- Pro: Deep insights, whale tracking
- Con: Expensive, complex

### Data Sources

**Price APIs**:
- CoinGecko API (most common)

**On-Chain Data**:
- Alchemy, Infura per Ethereum

**Exchange APIs**:
