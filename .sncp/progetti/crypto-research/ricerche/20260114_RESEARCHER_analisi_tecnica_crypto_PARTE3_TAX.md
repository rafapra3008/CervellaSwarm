## COST BASIS ALGORITHMS

### FIFO (First In First Out)
**Most Common in USA**

Example:
```
Buy 1 BTC @ $10K (Jan 1)
Buy 1 BTC @ $20K (Feb 1)
Sell 1 BTC @ $30K (Mar 1)

Cost basis = $10K (first purchased)
Gain = $30K - $10K = $20K
```

### LIFO (Last In First Out)
Example:
```
Buy 1 BTC @ $10K (Jan 1)
Buy 1 BTC @ $20K (Feb 1)
Sell 1 BTC @ $30K (Mar 1)

Cost basis = $20K (last purchased)
Gain = $30K - $20K = $10K
```

### HIFO (Highest In First Out)
**Tax Optimization**

Example:
```
Buy 1 BTC @ $10K (Jan 1)
Buy 1 BTC @ $25K (Feb 1)
Sell 1 BTC @ $30K (Mar 1)

Cost basis = $25K (highest cost)
Gain = $30K - $25K = $5K (minimize tax)
```

### Specific ID
**Manual Selection**

User specifies quale coin vendere.
Require detailed records.
